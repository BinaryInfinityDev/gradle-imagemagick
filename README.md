gradle-imagemagick
==================

Gradle tasks to easy integrate ImageMagick

## Install

```groovy
buildscript {
  repositories {
    jCenter()
  }

  dependencies {
    classpath 'com.eowise:gradle-imagemagick:0.5.2'
  }
}
```
