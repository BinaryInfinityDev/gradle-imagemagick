gradle-imagemagick
==================

Gradle tasks to easy integrate ImageMagick

## Install

```groovy
buildscript {
  repositories {
    mavenCentral()
  }

  dependencies {
    classpath 'com.eowise:gradle-imagemagick:0.4.0'
  }
}
```
