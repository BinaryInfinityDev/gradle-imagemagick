gradle-imagemagick
==================

Gradle tasks to easy integrate ImageMagick

## Install

```groovy
buildscript {
  repositories {
    maven {
      url: 'https://oss.sonatype.org/content/repositories/snapshots/'
  }

  dependencies {
    classpath 'com.eowise:gradle-imagemagick:0.3.0-SNAPSHOT'
  }
}
```
