gradle-imagemagick
==================

Gradle tasks to easily integrate ImageMagick

Install
-------

```groovy
buildscript {
  repositories {
    jCenter()
  }

  dependencies {
    classpath 'dev.binaryinfinity:gradle-imagemagick:0.0.1'
  }
}
```
